>* NVM node  
>* 安装truffle  

## （三）环境搭建  

`truffle init`改为`truffle unbox metacoin`可以产生几个文件: 
>* contracts  存放智能合约
>* migrations  存放帮助我们将智能合约部署到网络上的程序
>* test  存放用于测试智能合约的脚本  
>* truffle.js  存放一些配置信息  

## （四）主要命令  
>* truffle compile  
>* truffle migrate  
>* truffle console  
>* truffle test  
>* truffle unbox
